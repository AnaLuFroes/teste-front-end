Para rodar o projeto basta acessar a pasta vite-project

````
cd vite-project
````

e após acessar rodar a instalação das dependencias

````
npm i 
````

````
npm run dev
````

e acesse o site que vai estar rodando na porta 3000
pois no arquivo vite.config.ts está configurado a porta 3000

http://localhost:3000/
